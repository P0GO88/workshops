# Tech Fundamentals Vocabulary

## Word Bank

- hardware
- software
- file
- folder
- code
- program
- terminal

## Fill In the Blanks

1) A ____________________________ is a single item you can save on a computer, like a
   picture or a text note. \
2) A ____________________________ is a container that holds files so you can keep things
   organized. \
3) ____________________________ are the physical parts you can touch, like the keyboard,
   screen, or CPU. \
4) ____________________________ are the instructions that make hardware do useful work. \
5) ____________________________ is a set of precise instructions written for a computer
   to follow. \
6) When code is saved together to do a job, it becomes a ____________________________. \
7) A ____________________________ is a text-based way to talk directly to the computer
   using commands.
